# 📚 TUM Library Management System

This project is a simple object-oriented library management system developed as part of a university assignment at the Technical University of Munich (TUM).

## 🧠 Overview

The system models a basic library structure with the following hierarchy:

* **Library (TUMLibrary)**

  * Contains multiple sections
* **Section**

  * Represents a category (e.g., genre)
  * Contains multiple shelves
* **Shelf**

  * Stores multiple books
* **Book**

  * Represents a single book with metadata

## ⚙️ Features

* Add new sections to the library
* Add books to appropriate sections based on genre
* Prevent duplicate book IDs within a section
* Automatically organize books into shelves
* Sort books alphabetically by title
* Display the entire library structure

## 🏗️ Project Structure

```
src/
 └── de/tum/cit/fop/
     ├── Book.java
     ├── Section.java
     ├── Shelf.java
     └── TUMLibrary.java
```
